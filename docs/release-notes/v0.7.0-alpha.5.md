# Xuanhuan Earth on Minecraft 0.7.0-alpha.5

Development test build for Minecraft 26.2 / NeoForge 26.2.0.7-beta.

- Connects the existing initiation, practice, facility, active-technique, artifact and spirit-beast systems into one persistent six-step standalone journey.
- Advances journey state only from authoritative server success paths; failed actions, cooldowns, full recovery and uncollected automation outputs do not count.
- Handles normal and shift-click machine output collection without duplicate progress.
- Shows live journey completion and the next concrete action on the handbook start page in Chinese and English.
- Reconciles only initiation and practice from provable old-save progression.
- Grants 30 vanilla XP once when all six milestones are first completed.
- Adds pure regression coverage for ordering, idempotence, completion and unknown-bit sanitization.

Artifact: `xuanhuan-earth-on-minecraft-neoforge-26.2-0.7.0-alpha.5.jar`

Verification limit: source, resources and offline build are automated; client layout, entity interaction and full in-game journey completion still require an explicitly authorized smoke test.
