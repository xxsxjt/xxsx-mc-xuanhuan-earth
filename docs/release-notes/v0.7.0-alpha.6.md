# Xuanhuan Earth on Minecraft 0.7.0-alpha.6

Development test build for Minecraft 26.2 / NeoForge 26.2.0.7-beta.

- Adds a dedicated Image-2-generated mod icon instead of relying on a missing/default loader image.
- Uses a teal-jade Earth, golden cultivation ring and amethyst vein accent to identify the xuanhuan route.
- Registers the reviewed 512 x 512 PNG through `logoFile="logo.png"`.
- Preserves all alpha.5 gameplay, persistence, network and standalone journey behavior unchanged.

Artifact: `xuanhuan-earth-on-minecraft-neoforge-26.2-0.7.0-alpha.6.jar`

Verification limit: resource composition, metadata, build and artifact checks are automated; final appearance in the in-game mod list still requires an explicitly authorized client smoke test.
