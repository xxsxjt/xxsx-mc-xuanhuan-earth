# Xuanhuan Earth 0.7.0-alpha.8

## Naming migration

- Canonical public name: `Xuanhuan Earth / 玄幻地球`
- Canonical mod id and resource namespace: `xuanhuan_earth`
- Canonical repository and artifact prefix: `xuanhuan-earth`
- Canonical Java package: `com.xxsx.xuanhuanearth`
- Legacy `earth_online_xuanhuan:*` block, item, entity, block-entity, menu and feature ids resolve through NeoForge registry aliases.
- Legacy `earth_online_arcana.*` player data migrates to `earth_arcana.*` on first access.

## Target

- Minecraft 26.2
- NeoForge 26.2.0.7-beta
- JDK 25
- License: `AGPL-3.0-only`
- Artifact: `xuanhuan-earth-neoforge-26.2-0.7.0-alpha.8.jar`

This is an in-development prerelease. Build and static migration checks are complete; loading an existing alpha.7 world remains a runtime test item.
