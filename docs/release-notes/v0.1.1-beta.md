# Earth Online: Xuanhuan v0.1.1-beta

开发中测试版 / in-development beta.

## Target

- Minecraft: 26.2
- Loader: NeoForge 26.2.0.7-beta
- Java: 25
- Mod id: `earth_online_xuanhuan`
- Jar: `earth-online-xuanhuan-neoforge-26.2-0.1.1.jar`
- SHA256: `1BDC7801606D73C4477C9E848D5A811F8FAB13E6D91590A60226902776F92AF8`

## Highlights

- Adds the first chunk-level qi field runtime.
- Spirit vein nodes, spring stones, spirit soil, array cores, alchemy furnaces and talisman tables now contribute to local qi readings.
- Qi meditation now has cooldown and drains local chunk qi instead of being a free repeat-click recovery loop.
- Spirit compass, qi manual and xuanhuan environment blocks show readable field source and depletion feedback.
- Updates bilingual Chinese / English text for field readings, cooldowns and player guidance.

## Verification

- Built locally with `.\gradlew.bat build --no-daemon --offline`.
- Deployed to a NeoForge 26.2.0.7-beta client instance.
- Deployed jar hash matched the local build artifact.
- Jar contents were inspected for the new field classes and language resources.

## Known Limits

- Runtime smoke testing is still ongoing.
- Cross-chunk diffusion, persistent source registration, spirit plant growth hooks, spirit stone recovery and handbook GUI field pages are not implemented yet.
- Worldgen and field values may change between beta builds.
