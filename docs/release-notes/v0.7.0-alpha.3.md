# Xuanhuan Earth on Minecraft 0.7.0-alpha.3

Development test build for Minecraft 26.2 / NeoForge 26.2.0.7-beta.

- Adds a permanent Xuanhuan Earth Handbook separate from the Qi Guiding technique.
- Makes manuals unlock routes only; practice and recovery resolve in the cultivation panel or on a cushion.
- Adds independent level-1-to-10 XP tracks and permanent rewards for circulation, body tempering, fetal breathing and fasting.
- Adds mana-costed route skills with a configurable key and independent cooldown.
- Adds real attack-progress and sitting-state animation inputs for all five spirit-beast types.
- Keeps Earth on Minecraft, Earth Human and Fantasy Earth integrations optional.
