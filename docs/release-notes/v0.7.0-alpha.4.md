# Xuanhuan Earth on Minecraft 0.7.0-alpha.4

Development test build for Minecraft 26.2 / NeoForge 26.2.0.7-beta.

- Makes fasting/endurance progression reduce actual food and saturation loss.
- Makes body-tempering progression reduce entity-sourced combat damage and fall impact.
- Makes fetal-breath progression reduce underwater air consumption with a persistent multiplier.
- Gives Xuanhuan Earth ownership of shared body-adaptation consumption when both ecosystem mods are installed, preventing duplicate effects.
- Replaces write-only action NBT with server-broadcast player action payloads for cultivation, recovery, talismans and active techniques.
- Adds live unlock, level, XP and next-step guidance to the handbook start page.
- Synchronizes seated HUD stages with the authoritative server cooldown instead of entity age.
- Keeps all integrations optional and preserves standalone vanilla-material routes.

Artifact: `xuanhuan-earth-on-minecraft-neoforge-26.2-0.7.0-alpha.4.jar`

Verification limit: source, resources and offline build are automated; client layout, player poses and gameplay effects still require an explicitly authorized in-game smoke test.
