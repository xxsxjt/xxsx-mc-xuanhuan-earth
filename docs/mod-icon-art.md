# Mod icon art

- Provider/model: secured Image-2 workflow using `gpt-image-2`.
- Final resource: `neoforge-26.2/src/main/resources/logo.png`, 512 x 512 PNG.
- Visual identity: a faceted teal-jade Earth, natural green land, one golden cultivation ring, flowing qi and a restrained amethyst vein accent.
- Reference assets: the existing spirit compass and active alchemy furnace front were used only for palette and material language.
- Constraints: no text, no watermark, no generic MMO crest, and a silhouette that remains readable at 64 x 64.
- The original generated master remains in the local generated-image archive; the repository contains the reviewed downscaled release asset.
