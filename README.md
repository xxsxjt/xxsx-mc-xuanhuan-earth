# Xuanhuan Earth on Minecraft / 玄幻地球 on Minecraft

技术目录与兼容命名空间仍为 `earth_online_xuanhuan`，避免已有测试存档中的注册项失效。

这是一个可独立安装、独立完成入门与成长闭环的玄幻大型模组，面向“Minecraft 世界叠加玄幻规律”的玩法。安装《我的地球》《地球人》或《魔幻地球 on Minecraft》时会自动获得材料、人体与共享法力联动，但任何联动都不是游玩前置。

## 初始目标

- 不复刻真实生存难度；难度模拟由独立的 `earth_human` 方向处理。
- 不把玄幻内容塞进其他模组，也不要求玩家安装生态中的任何兄弟项目。
- 未安装《我的地球》时使用原版材料、独立世界生成和本模组资源完成全部基础路线；安装后额外识别现实矿物、岩石、植物、土壤、水文、金属和导能材料。
- 未安装《地球人》时修炼、恢复和属性成长仍正常工作；安装后再通过公开 API 增强疲劳与部位恢复。

## 初版可玩闭环

1. 玩家在现实地质和生态环境中发现灵性异常。
2. 用基础探测工具定位灵脉、灵泉、灵植、灵矿和异象区域。
3. 采集玄幻资源，但资源生成依托现实地形、岩性、水文和生态。
4. 用阵法/炉器/符具处理资源，产出丹材、符材、灵材、法器前置件。
5. 产物可回流到 MC 原版装备、药水、附魔、建筑和红石生态，也可给未来修行/人物 mod 使用。

## 当前状态

已生成第一版 NeoForge 26.2 独立模块：

- 模块：`neoforge-26.2`
- 目标：Minecraft 26.2 / NeoForge 26.2.0.7-beta / JDK 25
- 当前版本：`0.7.0-alpha.1`
- 产物：`xuanhuan-earth-on-minecraft-neoforge-26.2-0.7.0-alpha.1.jar`

当前测试版已有寻灵罗盘、区块灵气场、灵脉/灵泉/灵土、丹炉、符案、正式聚灵阵、JEI 分类、双语手册、功法、丹药、五种灵兽、三类居民与三种实体聚落；地球人身体联动保持可选。

`0.4.4` 进入大型模组阶段：新增可配置的修炼面板快捷键（默认 `K`），玩家不依赖蒲团也能在当前位置切换功法并行一周天。自由修炼使用 72% 基础效率；蒲团改为完整效率、额外聚灵、持续周天和更强恢复的可选辅助。面板状态、位置、冷却和收益均由服务端校验。

`0.7.0-alpha.1` 把生态切片推进到可测试闭环：五种独立灵兽支持亲和、契约、三种行动模式与持久化；散修商人、灵泉守护者和宗门执事使用原版商人容器；散修坊市、灵泉村和宗门边境前哨通过独立 Feature 与延迟聚落锚点生成，不替换原版村庄或修改原版生物。

`0.4.3` 修正符箓的能量逻辑：基础符箓是制作时在符案中吸纳大量环境灵气的预充能一次性载体，激发时只需 1 法力。成品不再允许通过合成台或灵墨捷径绕过符案充能，JEI、手册和双语提示会同时显示制作门槛与激发方式。

`0.4.2` 补齐统一法力消费、满状态防浪费、全部研习入口的冷却/场域消耗，并让设施识别《我的地球》的可选材料标签；手册会显示《我的地球》《地球人》和《魔幻地球 on Minecraft》的实时联动状态。人体恢复已迁移到公开 API，不再直接写入对方私有 NBT 或写死部位上限。

新增联动规则见：

- [共享法力值与玄幻修行协议](docs/shared-mana-and-cultivation.md)
- [灵气 / 魔力场与冥想系统规划](docs/arcana-field-and-meditation-plan.md)
- [Earth on Minecraft 生态联动契约](docs/ecosystem-integration-contract.md)
- [大型模组开发计划](docs/large-scale-development-plan.md)

当前玄幻入门物品为 `引气诀`：可用一块泥土、任意木板或任意石头合成。右键首次学习后，灵力路线会为统一法力值提供加成；之后可按默认 `K`（可在控制设置中修改）打开修炼面板。若同时安装《魔幻地球 on Minecraft》，两种路线的法力上限贡献相加；未安装时本模组仍完整可玩。
