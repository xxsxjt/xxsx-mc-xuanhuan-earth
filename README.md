# 地球 Online：玄幻

独立仓库名：`earth_online_xuanhuan`

这是 `地球 Online` 系列的玄幻分支，不并入本体。它面向“现实地球上叠加玄幻规律”的玩法：本体负责真实世界的自然来源、材料、工业和基础设施；本分支负责灵性资源、异象、秘境、灵物、术法和超自然生产链。

## 初始目标

- 不复刻真实生存难度；难度模拟由独立的 `earth_human` 方向处理。
- 不把玄幻内容塞进 `earth_online` 本体，避免污染现实内容核心。
- 与 `earth_online` 做软联动：读取矿物、岩石、植物、土壤、水文、金属、化工材料和电力体系，作为玄幻物质与法器/阵法/丹药的现实基底。
- 与 `earth_human` 未来做联动：体质、经脉、寿元、伤病、精神等人物层机制由 `earth_human` 提供，本 mod 只暴露灵性环境和修行资源接口。

## 初版可玩闭环

1. 玩家在现实地质和生态环境中发现灵性异常。
2. 用基础探测工具定位灵脉、灵泉、灵植、灵矿和异象区域。
3. 采集玄幻资源，但资源生成依托现实地形、岩性、水文和生态。
4. 用阵法/炉器/符具处理资源，产出丹材、符材、灵材、法器前置件。
5. 产物可回流到 MC 原版装备、药水、附魔、建筑和红石生态，也可给未来修行/人物 mod 使用。
